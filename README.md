
# My_Resume_ai

My_Resume_ai is an AI-powered resume builder inspired by resumevar.com. It allows users to create, upload, and tailor resumes with modern templates, live editing, and instant PDF downloads.


## Features

- AI-powered resume writing and suggestions
- Upload and enhance existing resumes (PDF, DOC, DOCX)
- Professional, ATS-friendly templates
- Real-time editing and live preview
- Instant PDF download
- Demo booking and FAQ
- Authentication (NextAuth.js, demo login: demo/demo123)

## Getting Started

1. Install dependencies:
	```bash
	npm install
	```
2. Run the development server:
	```bash
	npm run dev
	```
3. Open [http://localhost:3000](http://localhost:3000) (or the port shown in your terminal) in your browser.
